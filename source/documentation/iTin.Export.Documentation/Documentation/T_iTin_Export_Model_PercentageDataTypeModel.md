## Examples
The following example indicate that the content should be percentage data type. 
**XML**<br />
``` XML
<Style Name="PercentValue">
  <Content Color="Blue">
    <Percentage Decimals="1">
      <Error Value="0">
        <Comment Show="Yes">
          <Font Name="Comic Sans MS" Size="16" Bold="Yes" Italic="Yes" Underline="Yes" Color="Navy"/>
        </Comment>
      </Error>
    </Percentage>
  </Content>
</Style>
```
