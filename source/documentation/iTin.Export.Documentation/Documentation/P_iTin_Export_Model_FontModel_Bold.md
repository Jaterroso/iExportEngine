In the following example shows how create a new font. 
**XML**<br />


``` C#
var font = new FontModel
{
    Name = "Tahoma",
    Color = "Navy",
    Size = 8.0,
    Bold = YesNo.Yes,
    Italic = YesNo.Yes,
    Underline = YesNo.No
};
```
